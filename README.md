# BussPassSystem
# 🚌 Bus Pass Management System

A Java Swing based Bus Pass Management System with MySQL Database integration.

## 🚀 Features

- 🔐 User Login & Registration
- 🎫 Bus Pass Generation
- 🪪 Unique Token Verification System
- ⏳ Pass Expiry Timer
- 🔍 Search Pass using Token
- 💾 MySQL Database Connectivity
- 🎨 Java Swing GUI Interface
